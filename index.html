Exercise 14

What will the code below output to the console and why ?

1 + +"2" + "2" = "122". Plus symbol in both cases are interpreted as a concatenation.

1 + +"2" + "2" = "32". Plus symbol before the "2" is interpreted as a conversion to number (1 + +"2" = 1 + 2 = 3). And then the last plus is performed as a concatenation.

1 + "1" + "2" = "02". Minus symbol before the "1" is interpreted as a conversion to number (1 + +"1" = 1 - 1 = 0). And then the last plus is performed as a concatenation.

+"1" + "1" + "2" = "112". Plus symbol before the first "1" is interpreted as a conversion to number (+"1" = 1). And then the others plus symbols are performed as a concatenation.

+"A" - "B" + "2" = "NaN2". Subtraction between 2 letters is performed as a NaN. And then the other plus symbols are performed as a concatenation. So the result is "NaN2".

+"A" - "B" + 2 = "NaN2". Subtraction between 2 letters is performed as a NaN. And then the other plus symbols are performed as a addition. So the result is NaN.
